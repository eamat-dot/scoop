## 自分用メモ

jsonファイルのアップデートには `checkver`スクリプトを使用する

```powershell
ps> .\bin\checkver.ps1 <appname> -f
```

または

```powershell
ps> .\bin\checkver.ps1 <appname> -u
```

-f と -u のコマンドの違いはいまいちよく分からない
